# NCERT Mastery

NCERT Mastery is a lightweight, student-friendly web app for reading NCERT books and chapters in a clean, distraction-free interface. It provides a simple bookshelf experience, chapter-based PDF viewing, and a modern layout for focused study.

## What this project does
- Lets users browse NCERT books by subject and class
- Opens chapter PDFs in a responsive reader experience
- Offers a simple homepage and credits page
- Includes an Android APK download option for mobile use
